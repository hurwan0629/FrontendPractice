# 03. 함수 타입과 반환값

이번 단계 목표는 세 가지입니다.

1. 함수의 매개변수 타입을 자연스럽게 붙이기
2. 함수의 반환값 타입을 명확히 적기
3. 함수를 값처럼 다루는 기본 감각 익히기

## 1) 함수 타입은 왜 중요할까

실제 개발에서는 함수가 가장 자주 오가는 단위입니다.

- 버튼 클릭 함수
- 데이터 가공 함수
- 배열 메서드에 넘기는 콜백 함수
- 상태 업데이트 함수

함수 타입을 정확히 알면, 입력과 출력이 무엇인지 코드만 보고 바로 이해할 수 있습니다.

## 2) 매개변수 타입

```ts
function greet(name: string): string {
  return `Hello, ${name}`;
}
```

`name: string`은 함수가 문자열을 받는다는 뜻입니다.

## 3) 반환값 타입

```ts
function isAdult(age: number): boolean {
  return age >= 20;
}
```

반환값 타입은 함수가 "결국 무엇을 돌려주는가"를 보여줍니다.

처음 연습할 때는 반환값 타입도 직접 적는 습관이 좋습니다.

## 4) 함수를 변수에 담을 수도 있다

```ts
const subtract = (a: number, b: number): number => {
  return a - b;
};
```

함수 선언식뿐 아니라 화살표 함수에도 똑같이 타입이 붙습니다.

## 5) 콜백 함수 감각

배열 메서드에서 많이 보게 됩니다.

```ts
const numbers = [1, 2, 3];

const doubled = numbers.map((num: number): number => {
  return num * 2;
});
```

지금은 "함수도 타입 검사를 받는다" 정도만 이해하면 충분합니다.

## 6) 체크 포인트

이번 단계를 끝내면 아래를 혼자 할 수 있어야 합니다.

- 숫자를 받아 숫자를 반환하는 함수 만들기
- 문자열을 받아 문자열을 반환하는 함수 만들기
- 화살표 함수에 타입 붙이기
- 배열 `map` 안 콜백에 타입 붙이기
