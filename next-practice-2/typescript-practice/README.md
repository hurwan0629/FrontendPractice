# TypeScript Practice

TypeScript는 문법을 넓게 훑는 것보다, 작은 개념 하나를 이해하고 바로 손으로 써보는 방식이 더 빠르게 몸에 붙습니다.

이 폴더는 아래 순서로 진행합니다.

1. 기본 타입과 타입 표기
2. 객체 타입과 `type`
3. 함수 타입과 반환값
4. 유니온 타입과 리터럴 타입
5. 옵셔널 프로퍼티와 타입 추론
6. 배열, 객체, 함수 조합 연습
7. React props 타입으로 연결
8. `useState`와 이벤트 타입으로 연결

학습 방식은 항상 같습니다.

1. `concept.md`를 읽는다.
2. `practice.ts`의 TODO를 직접 채운다.
3. 타입 에러를 보고 스스로 수정한다.
4. 끝나면 다음 단계로 넘어간다.

첫 시작은 `01-basic-types` 입니다.
